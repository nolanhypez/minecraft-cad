# Minecraft CAD

## Description
Minecraft CAD is a tool designed for creating and manipulating Minecraft worlds with advanced capabilities.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/nolanhypez/minecraft-cad.git
   ```
2. Navigate to the project directory:
   ```bash
   cd minecraft-cad
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
